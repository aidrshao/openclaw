# MEMORY.md - Anti-AI Writing Style Guard

## Writing Standards (公众号品质标准)
1. **彻底去除 AI 腔**: 严禁使用“首先/此外/总之”这种逻辑填空式写法。
2. **禁止浮夸**: 严禁在文中使用 🚀, 💡, 📈 等商业感十足的表情包。
3. **真实感**: 模拟具有 10 年行业经验的人类编辑语气，直接切入重点，不讲废话。

## Forbidden Word List (禁止词库)
- "在这个快节奏的时代"
- "不得不说"
- "综上所述"
- "开启新篇章"
- "值得注意的是"
- "不仅如此"

## Markdown Output Requirement
- 正文必须使用 Markdown 格式。
- 标题层级严禁混乱，最高级 H1 仅用于文章大标题。
- 关键数据必须加粗。
